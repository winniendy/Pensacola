How to install smrtlink only tools:
Option 1: for all users
a. Download smrt link v11.0 from PacBio website (https://www.pacb.com/support/software-downloads/) 
     Introduction documents can alo be download from : 
         1) SMRT Link software installation (https://www.pacb.com/wp-content/uploads/SMRT_Link_Installation_v11.0.pdf)  
         2) SMRT Tools reference guide (https://www.pacb.com/wp-content/uploads/SMRT_Tools_Reference_Guide_v11.0.pdf)

b. Unzip and Install the software to your location:
                  ./smrtlink_11.0.0.146107.run --rootdir ./smrtlink_11 --smrttools-only

c. test if the command "pbmm2" working in command line.

Option 2: for HiperGator bphl-florida users
SMRT Link is already installed in /blue/bphl-florida/share/smrtlink_11. If you want to use its commands, the only thing needed to do is to add the SMRT LINK commands (smrtlinkcmds) directory in $PATH.
For temporarily adding the directory, run the command below in your terminal:
      export PATH=$PATH:/blue/bphl-florida/share/smrtlink_11/smrtlink_11/install/smrtlink-release_11.0.0.146107/bundles/smrttools/install/smrttools-release_11.0.0.146107/smrtcmds/bin
For permanently adding the directory, you need:
1) move to your home directory, such as "/home/<your name>"
2) open .bash_profile
3) add smrtlinkcmds directory in the end of PATH commandline, and then export PATH. For example:
        PATH=$PATH:/blue/bphl-florida/share/smrtlink_11/smrtlink_11/install/smrtlink-release_11.0.0.146107/bundles/smrttools/install/smrttools-release_11.0.0.146107/smrtcmds/bin
        export PATH
